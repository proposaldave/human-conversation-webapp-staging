# Human Conversation Web App Staging

This repository hosts the public, generated-data-only Human Conversation investor walkthrough.

- All people, events, dates, and results are synthetic.
- There are no live club integrations, real participant records, or real audio in this build.
- The walkthrough demonstrates the target workflow: choose, prepare, listen, remember, act, and prove.

Company context: [humanconversation.com](https://humanconversation.com/)
