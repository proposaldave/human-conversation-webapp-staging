# Human Conversation Web App Staging

This repository hosts the public, generated-data-only Human Conversation Lesson Studio investor demo.

- All people, events, dates, and results are synthetic.
- There are no live club integrations, real participant records, or real participant audio in this build.
- The demo shows a lesson moving from recording to AI-suggested human moments, coach approval, student sharing, referral, and attributable next-class revenue.

Company context: [humanconversation.com](https://humanconversation.com/)
